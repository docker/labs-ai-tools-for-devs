 1. **`list_api_resources`**: List the API resources available in the cluster
 1. **`list_cronjobs`**: List CronJobs in a namespace

## Use this MCP Server

```json
{
  "mcpServers": {
    "flux159-mcp-server-kubernetes": {
      "command": "docker",
      "args": [
        "run",
        "-i",
        "--rm",
        "mcpcommunity/flux159-mcp-server-kubernetes"
      ]
    }
  }
}
```

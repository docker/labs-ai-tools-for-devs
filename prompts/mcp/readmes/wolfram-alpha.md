## Use this MCP Server

```json
{
  "mcpServers": {
    "wolfram-alpha": {
      "command": "docker",
      "args": [
        "run",
        "-i",
        "--rm",
        "-e",
        "WOLFRAM_API_KEY",
        "mcp/wolfram-alpha"
      ],
      "env": {
        "WOLFRAM_API_KEY": "your-app-id"
      }
    }
  }
}
```

## Use this MCP Server

```json
{
  "mcpServers": {
    "e2b": {
      "command": "docker",
      "args": [
        "run",
        "-i",
        "--rm",
        "-e",
        "E2B_API_KEY",
        "mcp/e2b"
      ],
      "env": {
        "E2B_API_KEY": "YOUR_API_KEY_HERE"
      }
    }
  }
}
```

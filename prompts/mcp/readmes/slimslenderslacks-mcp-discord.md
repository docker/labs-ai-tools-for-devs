## Use this MCP Server

```json
{
  "mcpServers": {
    "slimslenderslacks-mcp-discord": {
      "command": "docker",
      "args": [
        "run",
        "-i",
        "--rm",
        "-e"
        "DISCORD_TOKEN"
        "mcpcommunity/slimslenderslacks-mcp-discord"
      ],
      "env": {
        "DISCORD_TOKEN": "YOUR_DISCORD_TOKEN"
      }
    }
  }
}
```

## Use this MCP Server

```json
{
  "mcpServers": {
    "tembo": {
      "command": "docker",
      "args": [
        "run",
        "-i",
        "--rm",
        "-e",
        "TEMBO_API_KEY",
        "mcp/tembo"
      ],
      "env": {
        "TEMBO_API_KEY": "your-tembo-apikey-here"
      }
    }
  }
}
```

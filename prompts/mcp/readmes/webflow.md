## Use this MCP Server

```json
{
  "mcpServers": {
    "webflow": {
      "command": "docker",
      "args": [
        "run",
        "-i",
        "--rm",
        "-e",
        "WEBFLOW_TOKEN",
        "mcp/webflow"
      ],
      "env": {
        "WEBFLOW_TOKEN": "your_api_token"
      }
    }
  }
}
```

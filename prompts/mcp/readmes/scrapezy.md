# Scrapezy MCP Server

## Use this MCP Server

```json
{
  "mcpServers": {
    "scrapezy": {
      "command": "docker",
      "args": [
        "run",
        "-i",
        "--rm",
        "-e",
        "SCRAPEZY_API_KEY",
        "mcp/scrapezy"
      ],
      "env": {
        "SCRAPEZY_API_KEY": "your_api_key"
      }
    }
  }
}
```
